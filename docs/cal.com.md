LanguageLab utilise le site https://cal.com pour programmer les séances entre les tuteurs et les étudiants.

Afin que l'intégration soit réussie, il est nécessaire de suivre les étapes suivantes:
 - Créer un compte sur https://cal.com.
 - Créer un nouveau type d'événement de 60 mins.
 - Créer un webhook avec l'URL https://languagelab.sipr.ucl.ac.be/api/v1/webhooks/sessions. Tous les triggers peuvent être gardés.
 - Copier le lien vers l'évenement.
 - Se rendre sur https://languagelab.sipr.ucl.ac.be/tutor/timeslots et y coller le lien.
